What is Tweak a Week?

01/03/14 10:17
Every week I will release a simple tweak on my Tweak a Week repo. Each tweak will come with Github source code and an in-depth explanation of the code in a blog post here. The point of this is for student, like myself, to learn how to make jailbreak tweaks. I want to do this because there is not a lot of information or guides online about making big tweaks. Ryan Petrich has tried his TweakWeek project, however it is outdated and has no explanation of the code. I will be assuming that you will already know how to create and compile a tweak and have Objective C knowledge. Thanks!
